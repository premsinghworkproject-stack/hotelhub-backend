# Example GraphQL Operations

## Create User
```graphql
mutation {
  createUser(input: {
    name: "Alice Johnson"
    email: "alice@example.com"
  }) {
    id
    name
    email
  }
}
```

## Create Hotel
```graphql
mutation {
  createHotel(input: {
    name: "Beach Paradise Resort"
    location: "Malibu, CA"
    price: 450.00
  }) {
    id
    name
    location
    price
  }
}
```

## Create Booking
```graphql
mutation {
  createBooking(input: {
    userId: "user-id-here"
    hotelId: "hotel-id-here"
    checkIn: "2024-07-15"
    checkOut: "2024-07-18"
  }) {
    id
    checkIn
    checkOut
    user {
      name
      email
    }
    hotel {
      name
      location
      price
    }
  }
}
```

## Get All Hotels (with pagination)
```graphql
query {
  hotels(limit: 5, offset: 0) {
    id
    name
    location
    price
  }
}
```

## Get Bookings by User
```graphql
query {
  bookingsByUser(userId: "user-id-here") {
    id
    checkIn
    checkOut
    hotel {
      name
      location
      price
    }
  }
}
```
